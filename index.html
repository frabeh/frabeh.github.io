Accordion Mobile & Web Apps

  • Client: Gist Digital
  • Date: Summer 2016
  • Service: UX/UI

I was the project manager and a designer for Accordion, an app to facilitate document sharing and communication within construction teams. I was tasked with leading a team through an iteration of the alpha mobile app and creating an analogous web app.

We first performed user testing to determine which industry segment would most benefit from the app. Using our research, we then updated the user flow and features list to make the app more useful to its target audience. We are in the process of working on both the mobile app (for both iOS and Android) and the web app.

User Testing

Accordion's original target audience was too broad, so I first led the team through user research and testing to determine which segment of the construction industry would most benefit from the app.

Below is a sample of the spreadsheet we used to collect our interview information and an interview outline (which included both user testing and market research). In between the pre- and post-app sections, we asked our users to click through the app and narrate their reactions to it.

Updated Features & Flow

We utilized the user feedback to create a more intuitive flow for the app. We also added user-requested features such as document folders, multiple chat channels per construction project, and an updated notification process.
