Accordion Mobile & Web Apps

I was the project manager and a designer for Accordion, an app to facilitate document sharing and communication. I was tasked with leading a team through an iteration of the alpha mobile app and creating a corresponding web app.

User Testing

Image #3 Image #4

Accordion's original target audience was unfocused, so I first led the team through user research and testing to determine which segment of the construction industry-- small construction companies-- would most benefit from the app.

I utilized the user feedback to create a more intuitive flow for the app. We also added some user-requested features such as document folders, multiple chat channels per construction project, and an updated notification process.
